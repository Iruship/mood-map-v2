## Run the backend

1. Go to the directory and Start a python venv
```bash
    cd backend
    python3 -m venv venv
    source venv/bin/activate
```

2. Install the dependencies
```bash
    pip install -r requirements.txt
```

3. Run the backend
```bash
    python3 main.py
```

## Run the frontend

1. Go to the directory and run npm install
```bash
    cd frontend
    npm install
```

2. Run the frontend
```bash
    npm run start
```
